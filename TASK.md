# Pyramid Solitaire H5 Game

## 概述
开发一个金字塔纸牌（Pyramid Solitaire）H5游戏，使用Phaser 3引擎，目标是上传到Y8等小游戏平台赚取广告分成。

## 游戏规则
- 28张牌摆成金字塔形（7行：1-2-3-4-5-6-7）
- 只有**未被遮挡的牌**（没有子牌覆盖）才可选
- 两张可选牌**点数之和=13**即可消除（如5+8, J+2, Q+A）
- K（=13）单独消除
- 剩余24张牌在发牌堆，每次翻1张到废牌堆
- 废牌堆顶牌也可参与配对
- 目标：消除金字塔中所有牌 = 赢
- 金字塔清空但还有牌未消除 = 输

## 技术要求
- **引擎**: Phaser 3 (CDN引入，单HTML文件)
- **输出**: 单个 index.html 文件（Y8上传要求）
- **尺寸**: 800x600 基础，响应式缩放
- **美术**: 代码绘制卡牌（圆角矩形+花色符号），不依赖外部图片
- **音效**: 暂不需要，后续可加

## 功能清单
1. ✅ 金字塔布局（7行28张牌，部分翻开部分暗牌）
2. ✅ 发牌堆 + 废牌堆
3. ✅ 点击选牌逻辑（高亮选中，配对消除）
4. ✅ 配对规则（和=13消除，K单独消除）
5. ✅ 翻牌（点击发牌堆翻1张到废牌堆）
6. ✅ 胜负判定（金字塔清空=赢，无法继续=输）
7. ✅ 计分系统（消除得分+连击加分+时间奖励）
8. ✅ 撤销功能（Undo最近一步）
9. ✅ 提示功能（高亮可消除的配对）
10. ✅ 新游戏/重开
11. ✅ 动画效果（消除动画、翻牌动画、胜利动画）
12. ✅ 移动端触控支持

## 卡牌设计
- 标准扑克牌52张（4花色×13点数）
- 花色：♠♥♦♣
- 点数：A(1) 2-10 J(11) Q(12) K(13)
- 卡牌尺寸：约70x100px
- 正面：白色圆角矩形 + 左上角点数花色 + 中间大花色
- 背面：蓝色圆角矩形 + 菱形图案
- 选中状态：金色边框发光
